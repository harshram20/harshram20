## 🚀 Featured Projects

### [GoSafe — Frontend](https://github.com/harshram20/Project)
> Most navigation apps optimize for speed. GoSafe optimizes for **safety** — scoring routes using real safety factors like crowd density, CCTV coverage, and emergency access points, visualized on an interactive map.

**What I built:**
- Generates up to 3 alternative routes ranked across 5 real safety factors with color-coded risk indicators (green / amber / red)
- Live GPS tracking with real-time location updates via the Geolocation API
- Smart autocomplete with real-origin/destination suggestions via Nominatim — debounced to prevent API spam
- One-tap Emergency SOS with deep linking to WhatsApp using GPS coordinates of saved contacts
- Route and modal state persisted across page loads using JWT (stored in `localStorage`)
- Waypoints decimated to max 200 points to prevent Leaflet rendering lag
- Fully responsive — mobile-first with collapsible bottom-sheet sidebar

**Design system built from scratch:** WCAG AA contrast ratios · CSS-only animations (zero JS animation libraries) · Design tokens for color, typography, and spacing · Responsive breakpoints at 768px / 480px

`React 18` `Vite` `Context API + useReducer` `React-Leaflet + OpenStreetMap` `Axios` `CSS Modules` `JWT` `Nominatim` `OSRM` `Overpass`

> ⚙️ Frontend complete. Backend (Node.js + PostgreSQL) in active development.
